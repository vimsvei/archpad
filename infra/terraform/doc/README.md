# Документация Terraform Infrastructure

Эта папка содержит всю документацию по Terraform инфраструктуре ArchPad.

## Структура документации

### Общие документы

- **README.md** (этот файл) - обзор документации
- **../README.md** - основной README с инструкциями по развертыванию

### Документация по развертыванию

#### Настройка инфраструктуры
- **DNS_SETUP.md** - настройка DNS записей для Traefik LoadBalancer
- **TLS_CERTIFICATES.md** - настройка TLS сертификатов через Traefik

#### Отладка и восстановление
- **DEBUG.md** - отладка проблем с развертыванием Traefik
- **RECOVERY.md** - восстановление после ошибок удаления Traefik
- **VAULT_DEBUG.md** - отладка проблем с развертыванием Vault

### Документация модулей

Каждый модуль имеет свой `README.md` в своей папке:
- `../init/README.md` - документация Init модуля
- `../deploy/01-traefik/README.md` - документация Traefik модуля
- `../deploy/02-vault/README.md` - документация Vault модуля
- `../deploy/2nd-step/README.md` - документация 2nd-step модуля

## Как использовать

Все документы находятся в этой папке для централизованного хранения. 
Каждый модуль также имеет свой `README.md` с инструкциями по использованию для быстрого доступа.
