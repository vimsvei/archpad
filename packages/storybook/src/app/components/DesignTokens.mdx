```css

padding: 0.75rem 1.5rem (py-3 px-6)
```

## üéØ Best Practices

<div style={{ padding: '1.5rem', background: '#0a0a0a', border: '1px solid #262626', borderLeft: '4px solid #22c55e', borderRadius: '4px', marginTop: '2rem' }}>
  <h3 style={{ margin: '0 0 1rem 0', color: '#fafafa' }}>Consistency Guidelines</h3>
  
  <ul style={{ margin: 0, paddingLeft: '1.5rem', color: '#a1a1aa' }}>
    <li>Always use CSS variables instead of hard-coded colors</li>
    <li>Maintain consistent spacing with gap/padding utilities</li>
    <li>Use semantic color names (foreground, muted-foreground, etc.)</li>
    <li>Apply opacity modifiers for hover states (bg-muted/20)</li>
    <li>Keep border radius consistent across similar components</li>
    <li>Use monospace font for code and technical data</li>
  </ul>
</div>
