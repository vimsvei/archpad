This folder contains files generated by GraphQL Code Generator.

- Run `pnpm --filter @archpad/portal gql:codegen`
- Uses `NODE_EXTRA_CA_CERTS=../../infra/traefik/certs/rootCA.pem` to trust local dev TLS.
- By default loads env from `packages/portal/.env` (via `dotenv/config`).
- If you want to load a different env file, run with `DOTENV_CONFIG_PATH=/path/to/.env`.
- Requires GraphQL endpoint env (one of):
  - `NEXT_PUBLIC_API_GRAPHQL_ENDPOINT` (preferred)
  - `API_GATEWAY_HOST` (will use `https://<host>/graphql/v1/graphql`)
  - `HASURA_HOST` (will use `https://<host>/v1/graphql`)
  - and optionally `HASURA_ADMIN_SECRET` / `HASURA_GRAPHQL_ADMIN_SECRET`

Do not edit generated files manually.
